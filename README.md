# V.O.I.C.E---Voice-Oriented-Intelligent-Command-Engine

An offline, AI-powered IoT automation system using **ESP32**, **OLED display**, and a **locally hosted chatbot** built with **Node.js** and **Ollama**.

## Project Overview

**V.O.I.C.E (Voice Oriented Intelligent Command Engine)** demonstrates the integration of **AI + IoT** for smart home automation.
